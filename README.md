# PHP

## Et kõik tööle hakkaks...

1. Klooni repo oma arvutisse ja hakka lahendama `src` kasutas olevat ülesannet
2. Ülesande kirjeldused on siin README failis.
3. Jooksuta teste, et näha, kas sa oled ülesande õigesti lahendanud

## Setup

Et kõik tööle hakkaks toimi järgnevalt:

Testide jooksutamiseks vajaliku tarvara saamiseks kirjuta terminali:
```bash
  composer install
```
Seejärel, et teste jooksutada kirjuta terminali:
```bash
  ./vendor/bin/pest
```

Kui ülesanded on lahendatud, pushi lahendused oma reposse seejärel
hakkavad automaatselt tööle testid, mis kontorollivad su tehtud tööd.

Hinde saad selle järgi kui mitu punkti saad. Erinevad testid annavad erineva arvu punkte.

10 puntki -> 5
8 punkti  -> 4
5 punkti  -> 3

Alla 5   -> MA
